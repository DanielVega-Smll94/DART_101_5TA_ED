# 🧑‍🎓 Carpeta ESTUDIANTE

Este es tu espacio personal para trabajar durante el curso **en vivo**.

Aquí puedes:

- ✍️ Escribir tus soluciones a las tareas asignadas.
- 👨‍🏫 Seguir el código que el profesor explica en clase y adaptarlo según sea necesario.
- 📁 Organizar tus archivos para practicar o reforzar lo aprendido.

---

## ✅ ¿Qué debes hacer aquí?

1. Durante la clase, puedes copiar o escribir el código que se trabaja en la carpeta `CURSO` y guardarlo aquí para practicar.
2. Cuando se asigne una tarea, búscala en la carpeta `TAREAS`.
3. Crea un archivo en esta carpeta con el **mismo nombre** que la tarea (por ejemplo, `tarea1.dart`).
4. Escribe tu solución y guarda los cambios.
5. Sube tu trabajo a GitHub con un *commit* y *push*.

## ❌ Qué NO debes hacer

- No edites las carpetas `CURSO` ni `TAREAS`.
- No cambies el nombre de esta carpeta.
- No subas archivos que no estén relacionados con el curso.

---

Este espacio es solo para ti. Úsalo para experimentar, equivocarte y aprender.  
¡Sigue el ritmo de las clases en vivo y construye tu propio camino en la programación!
